### Descripción
En esta tarea se presenta una implementación en C++ de una Lista Simplemente Enlazada, una Lista Doblemente Enlazada, un Árbol de Búsqueda Binaria, un Árbol Rojinegro y una Tabla de Dispersión. Para esto, se entregan los archivos _.hpp_, facilitados por el profesor, modificados con las implementaciones de las funciones de cada estructura.

Así mismo, se entrega un archivo 'main.cpp' que contiene el código con el que se ejecutaron los algoritmos para completar las demás partes de la tarea (mediciones de tiempo de ejecución), el cual puede ser utilizado para ejecutar las estructuras nuevamente si se desea.

De la misma forma, se entrega el reporte solicitado con el formato pertinente, el cual contiene el cuadro con los tiempos de ejecución de las operaciones de búsqueda y eliminación aleatoria y ordenada de cada estructura, sus respectivos gráficos de líneas, gráficos comparativos y un análisis escrito.

### Instrucciones de uso
Como se solicitó en las instrucciones, el código se puede compilar sencillamente con g++, de la siguiente forma (se asume que el compilador ya está instalado y que se utilizará el archivo 'main.cpp'):

### 1. En una terminal de Linux, se debe acceder al directorio donde se encuentran los archivos de código fuente del programa (source, en este caso).

### 2. Se debe escribir el siguiente comando:

    g++ main.cpp -o programa_estructuras

Esto creará el archivo ejecutable del programa llamado 'programa_estructuras'.

### 3. Se debe escribir el comando:

    ./programa_estructuras

De esta forma, el programa se ejecutará correctamente.
